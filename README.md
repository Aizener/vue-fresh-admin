### 关于vue-fresh-admin介绍
这是一个基于Vue3练手的一个后台管理项目，基于实验性特性：[响应性语法糖](https://cn.vuejs.org/guide/extras/reactivity-transform.html#reactivity-transform)来编写。  

对于该项目的命名，重点在于`fresh`这个词： 
该词作形容词时意为"新鲜的；清新的；淡水的；无经验的"，以此来表达这个项目的简洁性。
